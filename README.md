## 🌟 About The Project

**NOVAInvoice** is a sleek and modern **invoice generator web application** built using **HTML, CSS, and JavaScript**.

It allows users to create, customize, and manage invoices with real-time calculations and a professional layout — making it perfect for freelancers, developers, and small businesses.

---

## ✨ Key Features

- 🧾 Dynamic invoice creation  
- ➕ Add / remove items easily  
- 💰 Real-time price calculation  
- 📊 Automatic total & tax calculation  
- 📄 Clean printable invoice layout  
- 📥 Download / print invoice  
- 📱 Fully responsive design  
- ⚡ Smooth and interactive UI
